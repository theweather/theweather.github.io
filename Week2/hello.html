Some notes from basic Javascript stuff:
  • Variable names are case sensitive.
  • Variable names must begin with a letter or an underscore.
  • Name variables in such a way as to increase the visibility/understanding of the users.
  • == is Equal Value
  • === is Equal Value and Type
  • != is Does Not Equal
  • !== is Does Not Equal Value or Type
  • > is Greater Than
  • < is Less Than
  • >= is Greater Than or Equal To
  • <= is Less Than or Equal To
  • && is AND
  • || is OR
